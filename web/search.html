Ceccun relies on Javascript

Javascript allows Ceccun to build interactive experiences and efficient single-page apps that work perfectly in your web browser.

In some web browsers, Javascript is disabled which means that Ceccun needs to fallback to using old and outdated solutions for critical services.

You should turn on Javascript or switch to another browser to get the best experience.

All of Ceccun's vital services, including search, will have a Javascript-free version available.
